# scan-py documentation website

This repository is prepared for GitHub Pages using the `docs/` folder as the website source.

## Publish on GitHub Pages

1. Commit and push these files to the `main` branch.
2. In the GitHub repository, open **Settings** > **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Set the branch to `main` and the folder to `/docs`.
5. Save the settings.

After GitHub finishes building the site, it will be available at:

```text
https://prabashoka.github.io/scan-py-documentation/
```

The website home page is `docs/index.md`. Add more Markdown pages to `docs/` and link to them from the navigation block at the top of each page.
